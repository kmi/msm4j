Project Modules

This project has declared the following modules:

Name Description
MSM4J Java Object Model MSM4J Vocabulary provides a Java object model for the manipulation of service descriptions in terms of the Minimal Service Model vocabulary.
MSM4J I/O MSM4J I/O provides basic support for reading and writing service descriptions expressed in terms of the Minimal Service Model.
MSM4J hRESTS Transformer hRESTS Transformer provides support for transforming hRESTS service descriptions into MSM.
MSM4J OWL-S Transformer OWL-S Importer transformer support for transforming OWL-S service descriptions into Minimal Service Model.
SAWSDL Transformer SAWSDL Transformer provides support for transforming SAWSDL service descriptions into Minimal Service Model.
MSM4J WSC 2008 Transformer MSM4J WSC 2008 Transformer provides support for transforming service descriptions in terms of the proprietary format of the WSC 2008 contest into Minimal Service Model.
MSM4J Swagger Transformer Swagger Transformer provides support for transforming Swagger descriptions into Minimal Service Model.
